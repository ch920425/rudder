Rudder 的稳定版本对应用户默认安装路径：

```sh
npx @rudderhq/cli@latest start
```

这个页面只记录稳定版本。Canary 会更频繁地用于验证和 Desktop 资产测试；如果需要查看预发布历史，可以看 [GitHub Releases](https://github.com/Undertone0809/rudder/releases)。

## v0.3.0

发布时间：2026-06-04

[GitHub Release](https://github.com/Undertone0809/rudder/releases/tag/v0.3.0)

### 重点变化

- 将 Library 提升为组织文件夹、项目文件、内置 skill references 和 workspace launcher 的主导航入口。
- 在 chat、comments、markdown editor 和 Milkdown 中加入 Library-folder mention，方便从输入流里直接引用共享上下文。
- 改善 issue detail 的阅读体验，包括 sticky sidebar、更清晰的 status icon、空标题创建状态、继承 sub-issue workspace 校验和 run-duration 展示。
- 收紧 Messenger 和 chat 的交互，包括 sidebar 密度控制、copy-message tooltip、相对时间和 mention 搜索结果。
- 增加 heartbeat instruction scene、继承 project context 和 calendar event detail 的运行时/Calendar 保护。
