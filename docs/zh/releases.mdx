Rudder 的稳定版本对应用户默认安装路径：

```sh
npx @rudderhq/cli@latest start
```

这个页面只记录稳定版本。Canary 会更频繁地用于验证和 Desktop 资产测试；如果需要查看预发布历史，可以看 [GitHub Releases](https://github.com/Undertone0809/rudder/releases)。

## v0.3.5

发布时间：2026-06-18

[GitHub Release](https://github.com/Undertone0809/rudder/releases/tag/v0.3.5)

### New Features

- 增强 Messenger 分组、默认 tab、chat title 生成，以及 agent run chat link 展示。
- 增加 Library 的 PDF preview、editable CSV、HTML preview 和更快的 path-hinted entry link 打开体验。
- 增加 Codex subscription cost controls、默认 cost estimates，以及 user activity ledger CLI 覆盖。

### Improvements

- 改善 Codex、Claude、OpenCode、Pi 和 custom provider model 的 runtime onboarding 与 diagnostics。
- 改善 cost、analytics、automation、transcript 和 preview surfaces，适配更高频的 operator workflow。

### Bug Fixes

- 修复 chat finalization、title preservation、comment wake intent、organization cache isolation、release cleanup 和 CI stability 问题。
