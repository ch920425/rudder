## Highlights

- Added followed automation issues so recurring automations can surface their
  associated issue work directly in operator workflows.
- Improved Messenger navigation and issue context with agent-grouped sidebars,
  optimistic unread clearing, stale workspace cleanup on project switches, and
  stronger sidebar query caching.
- Added inline chat message editing and tightened mention menus so skill and
  composer suggestions stay anchored and readable during editing.
- Improved Markdown rendering for patch-style code blocks, code-copy behavior,
  and editor chrome across Library and chat surfaces.
- Refined Desktop and shell UI details with cross-platform dev startup fixes,
  Windows-scoped rail sizing, updated rail icons, and tighter workspace
  launcher presentation.
- Expanded regression coverage for automation follow-up issues, Messenger
  contracts, chat editing, mention insertion, markdown patch rendering,
  project-switch cleanup, and Desktop rail behavior.

## Install

```sh
npx @rudderhq/cli@latest start
```

After the persistent CLI is installed, `rudder start` uses the same CLI surface
when it resolves to the same version.

## Notes

This release promotes the `0.3.2` package line to the stable npm `latest`
dist-tag. Desktop assets are attached to the `v0.3.2` GitHub Release and
verified by checksum before installation.
