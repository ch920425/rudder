## New Features

- Added Feishu integration setup sessions, inbound callbacks, outbound responses,
  source badges, and read-only Feishu-bound chats with explicit fork handoff for
  operator replies.
- Added chat conversation forks, Messenger custom groups, pinned groups,
  generated group titles, group icons, and active chat actions aligned with the
  Messenger sidebar.
- Added richer Library and runtime surfaces, including organization skills in
  Library, workspace backup version downloads, syntax-highlighted editing,
  PDF/CSV previews, and runtime startup context bundles.

## Improvements

- Improved local runtime reliability for Codex, Claude, Cursor, OpenCode, Pi,
  and embedded Postgres by isolating runtime homes, preserving operator home
  access where needed, and tightening auth and platform diagnostics.
- Improved Messenger, chat, comment, and issue workflows with clearer mention
  rendering, grouped-row read handling, safer composer references, queue
  steering, and stronger sidebar loading caps.
- Improved maintainer workflows with product logic registry guardrails,
  release changelog taxonomy, lifecycle routing guidance, acceptance verifier
  skill coverage, and more explicit bundled-skill documentation.
