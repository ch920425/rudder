## Highlights

- Improved global navigation with scoped command-palette search, loading feedback,
  and primary-rail destination memory so operators return to the exact section
  or issue detail they were using.
- Polished renderable links and mention surfaces, including issue-comment hover
  previews, entity-aware preview cards, website link chips, and clearer issue
  status icons in Markdown and editors.
- Expanded project identity controls with a broader semantic icon set, larger
  project glyphs, aligned project identity triggers, and preserved project icon
  mention chips.
- Added mention-aware proposal review feedback so reviewers can reference agents
  and skills directly while deciding on chat-created proposals.
- Hardened release maintenance by cleaning obsolete canary GitHub Releases and
  tags after a stable publish while preserving the active npm canary line.
- Stabilized release and CI coverage around heartbeat teardown, chat proposal
  tests, renderable-link behavior, shortcut platform detection, and release
  canary cleanup planning.

## Install

```sh
npx @rudderhq/cli@latest start
```

After the persistent CLI is installed, `rudder start` uses the same CLI surface
when it resolves to the same version.

## Notes

This release promotes the `0.3.4` package line to the stable npm `latest`
dist-tag. Desktop assets are attached to the `v0.3.4` GitHub Release and
verified by checksum before installation.
