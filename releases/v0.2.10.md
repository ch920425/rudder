## Highlights

- Promoted the Library workspace model for organization files, project
  resources, agent instructions, and bundled skill references.
- Improved Library file actions, editor tabs, drag behavior, mention tokens, and
  protected managed entries so operators can edit shared knowledge without
  breaking runtime-owned files.
- Tightened Messenger and issue attention behavior, including pagination,
  self-authored comment filtering, and clearer issue update cards.
- Improved agent run and heartbeat surfaces with better run rows, timestamps,
  avatars, skill analytics, and timer-admission checks.
- Added richer markdown mention support across comments, chat, Library files,
  and editor surfaces.
- Polished Desktop and local runtime paths around bundled plugin assets,
  release staging, and user-facing runtime errors.
- Fixed CI-facing UI test drift around markdown rendering, linked run output,
  and workspace browser router mocks.

## Install

```sh
npx @rudderhq/cli@latest start
```

After the persistent CLI is installed, `rudder start` uses the same CLI surface
when it resolves to the same version.

## Notes

This release promotes the `0.2.10` package line to the stable npm `latest`
dist-tag. Desktop assets are attached to the `v0.2.10` GitHub Release and
verified by checksum before installation.
