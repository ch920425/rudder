# Rudder v0.2.7

Released: 2026-05-27

Rudder 0.2.7 is a product and control-plane release focused on richer
organization context, faster messenger and sidebar data paths, broader chat and
automation workflows, refreshed onboarding documentation, and a more complete
internal UI lab for product iteration.

## Highlights

- Added organization intelligence profiles so teams can capture reusable
  operating context and expose it through organization settings, import/export,
  validation, and server APIs.
- Expanded chat, messenger, and automation workflows with multi-select
  ask-user questions, chat-created automations, agent attribution navigation,
  proposal review improvements, attachment previews, and more reliable
  automation chat output handling.
- Improved control-plane performance with cheaper messenger summaries, sidebar
  badge aggregation, monthly cost spend rollups, approval summary indexing, and
  a reproducible performance baseline harness.
- Refined agent and run surfaces with run skill filters, fallback avatars,
  working-issue counts, recent issue behavior, and a broader internal UI lab for
  validating components and interaction states.
- Hardened operational reliability around scheduled database backups,
  heartbeat closeout behavior, release workflow serialization, local operations
  documentation, and maintainer skill routing.
- Refreshed README and public documentation across English and Chinese
  onboarding, runtime configuration, agent creation, concepts, issue lifecycle,
  and first-organization guides.

## Install

```sh
npx @rudderhq/cli@latest start
```

After the persistent CLI is installed, `rudder start` uses the same CLI surface
when it resolves to the same version.

## Notes

This release promotes the `0.2.7` package line to the stable npm `latest`
dist-tag. Desktop assets are attached to the `v0.2.7` GitHub Release and
verified by checksum before installation.
