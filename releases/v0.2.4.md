# Rudder v0.2.4

Released: 2026-05-18

Rudder 0.2.4 is a patch release focused on Desktop update safety, agent CLI
coverage, issue and automation workflow polish, and denser board/chat surfaces
after the 0.2.3 line.

## Highlights

- Hardened Desktop and CLI startup/update behavior with safer quit handoff,
  packaged CLI stdout flushing, cached portable asset handling, workspace
  launcher icon fixes, and runtime install coverage.
- Expanded the agent-facing CLI contract with private skill creation, richer
  issue search/comment/document commands, and synchronized bundled skill
  references for the `rudder` and `rudder-create-agent` skills.
- Improved issue workflows with starred issues in the sidebar, activity
  principal filters, reviewer and approval route coverage, comment persistence
  signals, ownership boundary fixes for mentions, and approval-blocked issue
  reactivation.
- Refined automation, calendar, and activity UI with clearer archive/delete
  terminology, projected heartbeat selector coverage, autopilot layout polish,
  and denser automation creation/detail screens.
- Polished chat and board presentation with expandable long message previews,
  chat response quality gating, better empty/project headings, constrained card
  metadata, and improved agent avatar handling.
- Added a knowledge base control-plane proposal and updated release-adjacent
  docs/tests for the new agent and CLI surfaces.

## Install

```sh
npx @rudderhq/cli@latest start
```

After the persistent CLI is installed, `rudder start` uses the same CLI surface
when it resolves to the same version.

## Notes

This release promotes the `0.2.4` package line to the stable npm `latest`
dist-tag. Desktop assets are attached to the `v0.2.4` GitHub Release and
verified by checksum before installation.
